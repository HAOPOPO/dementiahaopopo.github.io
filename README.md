# 回家的路 - 失智症認知迷宮遊戲

一個互動式網頁遊戲，透過迷宮問答的方式，幫助玩家認識失智症相關知識。

## 遊戲簡介

75歲的郝婆婆患有失智症，今天她出門散步，卻在回家的路上喪失了方向感而迷路。玩家需要透過回答30道關於失智症的是非題，引導郝婆婆走過迷宮，順利回到家中。

## 遊戲特色

- **互動式迷宮**：使用HTML5 Canvas繪製的迷宮地圖
- **30道問答題**：涵蓋失智症預防、症狀、照護等知識
- **視覺化移動**：郝婆婆會根據答題結果在迷宮中移動
- **即時反饋**：答題後立即顯示正確答案和解釋
- **成績統計**：遊戲結束後顯示完整作答紀錄

## 如何使用

### 線上遊玩

直接訪問 GitHub Pages：[遊戲連結]（部署後填入）

### 本地執行

1. 下載專案
```bash
git clone https://github.com/你的用戶名/dementia-maze-game.git
```

2. 開啟遊戲
- 直接用瀏覽器開啟 `index.html` 檔案即可

### 部署到GitHub Pages

1. 將專案推送到GitHub
2. 進入Repository設定
3. 找到 "Pages" 選項
4. 在 "Source" 選擇 `main` 分支
5. 點擊 "Save"
6. 等待幾分鐘後，遊戲就會部署在 `https://你的用戶名.github.io/dementia-maze-game/`

## 技術細節

- **純前端實作**：HTML + CSS + JavaScript
- **無需後端**：所有邏輯都在瀏覽器端執行
- **響應式設計**：支援桌面和行動裝置
- **無外部依賴**：不需要安裝任何套件或框架

## 遊戲規則

1. 從起點出發，回答第2題開始
2. 每題有「是」和「否」兩個選項
3. 根據答案，郝婆婆會移動到不同的題號
4. 答錯可能會碰壁，需要重新開始
5. 答對所有題目並到達終點（房屋）即獲勝

## 教育意義

本遊戲旨在透過互動方式傳遞失智症相關知識，包括：
- 失智症的預防方法
- 早期警訊識別
- 常見迷思破解
- 照護正確觀念
- 危險因子認識
